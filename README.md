# Insert Makerとは？
Excel形式のファイルからDBのInsert文を作成するWebサービスです。
また当初はExcelファイルのみサポートでしたが、CSVファイルからもInsert文を作成できるよう機能追加しました。  
<br>
さらにExcel・CSVファイルが無くてもInsert文を生成できるよう機能も追加しました。
<br><br>
↓以下がWEBサイトのURLになります
